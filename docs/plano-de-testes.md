# Plano de Testes

## 1. Objetivo

Validar as funcionalidades do sistema de Controle de Produtos, garantindo que os fluxos de acesso e gerenciamento de produtos funcionem conforme esperado.

## 2. Escopo

### Funcionalidades contempladas

* Login
* Cadastro de produtos
* Edição de produtos
* Exclusão de produtos

### Fora do escopo

* Testes de performance

## 3. Tipos de testes executados

* Testes Funcionais
* Testes de Interface (UI)
* Testes Automatizados

## 4. Ambiente

* Sistema Operacional: Windows 11
* Linguagem: Java 21
* Framework de Teste: JUnit 5
* Automação Web: Selenium WebDriver
* Gerenciador de Dependências: Maven
* Navegador: Google Chrome

## 5. Critérios de Aprovação

* Todos os cenários críticos aprovados.
* Nenhum defeito crítico ou bloqueante aberto.
